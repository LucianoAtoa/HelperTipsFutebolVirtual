  <how-to-verify>
1. Start the dashboard: `python -m helpertips.dashboard`
2. Open http://localhost:8050 in a browser

**Check layout (D-01, D-03, DASH-01):**
- Page has dark background (DARKLY theme)
- Layout is vertical: KPI cards top, filters below, charts in 2-column grid, table at bottom
- On mobile/narrow window: columns stack vertically (Bootstrap responsive)

**Check KPI cards (DASH-02):**
- 5 cards show numbers from the database
- Green color for Greens, red for Reds, yellow/amber for Pendentes, blue for Win Rate

**Check filters (DASH-04, DASH-05, D-09, D-10, D-11, D-12):**
- Select a liga from dropdown — all stats, charts, and table update immediately (no Apply button)
- Select an entrada — same reactive update
- Select a date range — data filters by date
- Click Reset — all filters clear, full data shown

**Check ROI simulation (DASH-03, D-05, D-06, D-07, D-08):**
- Default shows R$10 stake, 1.90 odd
- Change stake to R$20 — profit values update
- Toggle "Com Gale" on — values change (should show higher risk/cost)
- Toggle "Com Gale" off — values return to stake fixa mode

**Check charts (D-02):**
- Both charts have dark background (plotly_dark)
- Bar chart shows GREEN/RED/Pendente counts
- Win Rate per Liga shows horizontal bars with % labels

**Check history table (DASH-06, DASH-07, D-13, D-14, D-15, D-16, D-17):**
- Table has dark theme (alpine-dark)
- Shows 6 columns: Liga, Entrada, Horario, Resultado, Placar, Tent.
- 20 rows per page with pagination controls
- Click column header to sort
- Rows without resultado have amber/yellow background highlight

**Check auto-refresh:**
- Wait 60 seconds or check that dcc.Interval is present in page source
