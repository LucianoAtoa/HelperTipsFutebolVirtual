**4. NOVA funcao `_resolve_periodo` (per D-04, D-05):**
```python
def _resolve_periodo(periodo: str | None, custom_start=None, custom_end=None):
    """Converte selecao de periodo em (date_start, date_end) ISO strings."""
    today = date.today()
    if periodo == "hoje":
        return str(today), str(today)
    elif periodo == "semana":
        start = today - timedelta(days=today.weekday())
        return str(start), str(today)
    elif periodo == "mes":
        start = today.replace(day=1)
        return str(start), str(today)
    elif periodo == "mes_passado":
        first_this_month = today.replace(day=1)
        last_month_end = first_this_month - timedelta(days=1)
        last_month_start = last_month_end.replace(day=1)
        return str(last_month_start), str(last_month_end)
    elif periodo == "personalizado":
        return custom_start, custom_end
    else:  # "toda_vida" ou None
        return None, None
```

Corpo do callback:
1. `date_start, date_end = _resolve_periodo(periodo, custom_start, custom_end)`
2. Sanitizar stake/odd: `stake = float(stake or 10)`, `odd = float(odd or 1.65)`
3. Converter mercado dropdown: `entrada = mercado if mercado else None` (usar `entrada` param, nao `mercado_id` — Pitfall 2 do RESEARCH)
4. `conn = get_connection()` + try/finally com `conn.close()`
5. `stats = get_filtered_stats(conn, liga=liga, entrada=entrada, date_start=date_start, date_end=date_end)`
6. `history = get_signal_history(conn, liga=liga, entrada=entrada, date_start=date_start, date_end=date_end)`
7. KPIs basicos: total = `stats["total"]`, winrate = `f"{stats['greens']/(stats['greens']+stats['reds'])*100:.1f}%" if (stats['greens']+stats['reds']) > 0 else "\u2014"`
8. P&L principal: `roi_principal = calculate_roi(history, stake, odd, gale_on)`, `pl_principal = roi_principal["profit"]`
9. P&L complementar (per D-10): se mercado especifico, buscar `get_complementares_config(conn, _entrada_para_slug(mercado))` e `calculate_roi_complementares(history, comp_config, stake, gale_on)`. Se "Todos" (mercado vazio), pl_comp = 0.
10. P&L total: `pl_total = pl_principal + pl_comp`
11. ROI total: calcular baseado em total_invested combinado. Se total_invested == 0, roi = 0.
12. Formatacao P&L: `f"R$ {pl_total:+.2f}"`, classe `"card-text fw-bold text-success"` se > 0, `"card-text fw-bold text-danger"` se < 0, `"card-text fw-bold text-muted"` se == 0.
13. Formatacao ROI: `f"{roi_total:+.1f}%"`, mesma logica de cor.
14. Streaks: `streaks = calculate_streaks(history)`, green = `f"{streaks['max_green']}x" if streaks['max_green'] > 0 else "\u2014"`, red similar.
15. Liga options: `[{"label": v, "value": v} for v in get_distinct_values(conn, "liga")]`
16. History rowData: formatar para AG Grid (manter formato existente — data_hora, liga, entrada, time_casa, time_fora, resultado, tentativa, placar).
17. Return tuple com todos os 10 Outputs na ordem.
