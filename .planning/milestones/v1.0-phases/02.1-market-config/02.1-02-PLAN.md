  <implementation>
    Adicionar ao final de helpertips/queries.py (antes de nenhum import adicional necessario):

    1. `_parse_placar(placar: str) -> tuple[int, int] | None` — split por "-", int(), retorna None em qualquer falha.

    2. `_REGRA_VALIDATORS: dict[str, Callable[[int, int], bool]]` — dict com 7 lambdas:
       - "over_3_5": lambda casa, fora: (casa + fora) > 3.5
       - "over_5_plus": lambda casa, fora: (casa + fora) >= 6
       - "empate_3_3_4_4": lambda casa, fora: (casa + fora) in (6, 8) and casa == fora
       - "gols_casa_4": lambda casa, fora: casa == 4
       - "gols_fora_4": lambda casa, fora: fora == 4
       - "gols_casa_5_plus": lambda casa, fora: casa >= 5
       - "gols_fora_5_plus": lambda casa, fora: fora >= 5

    3. `validar_complementar(regra: str, placar: str | None, resultado_principal: str | None) -> str | None`:
       - Se resultado_principal is None -> return None
       - Se resultado_principal == "RED" or placar is None -> return "RED"
       - Parsear placar. Se None -> return "RED"
       - Buscar validator no dict. Se None -> return "RED"
       - Return "GREEN" if validator(casa, fora) else "RED"

    Import `from typing import Callable` no topo do arquivo.
  </implementation>
