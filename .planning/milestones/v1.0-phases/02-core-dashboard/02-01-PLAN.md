**Create `helpertips/queries.py`** with these functions:

1. `get_filtered_stats(conn, liga=None, entrada=None, date_start=None, date_end=None) -> dict`
   - Build WHERE clause dynamically with parameterized queries (use `%s` placeholders, never f-strings for values)
   - Base condition: `1=1`
   - If liga: `AND liga = %s`
   - If entrada: `AND entrada = %s`
   - If date_start: `AND received_at::date >= %s::date` (per Pitfall 4 — DatePickerRange returns ISO strings)
   - If date_end: `AND received_at::date <= %s::date`
   - SQL: `SELECT COUNT(*) AS total, COUNT(*) FILTER (WHERE resultado = 'GREEN') AS greens, COUNT(*) FILTER (WHERE resultado = 'RED') AS reds, COUNT(*) FILTER (WHERE resultado IS NULL) AS pending FROM signals WHERE {where}`
   - Return dict: `{"total": int, "greens": int, "reds": int, "pending": int}`

2. `get_signal_history(conn, liga=None, entrada=None, date_start=None, date_end=None, limit=500) -> list[dict]`
   - Same WHERE clause builder pattern
   - SQL: `SELECT liga, entrada, horario, resultado, placar, tentativa, received_at FROM signals WHERE {where} ORDER BY received_at DESC LIMIT %s`
   - Return list of dicts (use `cur.description` for column names or explicit dict construction)
   - Per D-14: default ordering is received_at DESC
   - Per D-15: return all 6 visible columns: liga, entrada, horario, resultado, placar, tentativa

3. `get_distinct_values(conn, field: str) -> list[str]`
   - Validate `field` is in allowed set: `{"liga", "entrada"}` — raise ValueError otherwise (SQL injection prevention)
   - SQL: `SELECT DISTINCT {field} FROM signals WHERE {field} IS NOT NULL ORDER BY {field}`
   - Return flat list of strings

4. `calculate_roi(signals: list[dict], stake: float, odd: float, gale_on: bool) -> dict`
   - Pure Python, NO database access
   - For each signal with resultado not None:
     - `tentativa = signal.get("tentativa") or 1`
     - **Stake Fixa mode (gale_on=False):** effective_stake = stake for every signal
     - **Gale mode (gale_on=True):** Per D-06 and Pitfall 5:
       - GREEN at tentativa N: invested = sum of stakes for attempts 1..N = stake * (2^0 + 2^1 + ... + 2^(N-1)) = stake * (2^N - 1). Profit = stake * 2^(N-1) * (odd - 1) - stake * (2^(N-1) - 1) [lost attempts before green]
       - RED (all 4 failed): invested = stake * (2^0 + 2^1 + 2^2 + 2^3) = stake * 15. Loss = -stake * 15
       - Simpler formulation: for any signal, accumulated_stake = stake * (2^tentativa - 1). If GREEN: net = stake * 2^(tentativa-1) * (odd - 1) - stake * (2^(tentativa-1) - 1). If RED: net = -accumulated_stake.
     - Skip signals with resultado=None (pending)
   - Return: `{"profit": float, "roi_pct": float, "total_invested": float}`
   - roi_pct = (profit / total_invested * 100) if total_invested > 0 else 0.0
