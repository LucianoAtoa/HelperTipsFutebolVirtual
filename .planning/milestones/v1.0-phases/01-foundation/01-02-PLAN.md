   - `import re` (only stdlib — NO imports from db, store, telethon)

   - Define regex patterns:
     - `LIGA_PATTERN = r'(?:⚽\s*)?LIGA\s*:\s*(.+?)(?:\n|$)'` (case-insensitive)
     - `ENTRADA_PATTERN = r'(?:🎯\s*)?Entrada\s*:\s*(.+?)(?:\n|$)'` (case-insensitive)
     - `HORARIO_PATTERN = r'(?:⏰\s*)?Hor[aá]rio\s*:\s*(\d{1,2}:\d{2})(?:\n|$)'` (case-insensitive)
     - `RESULTADO_PATTERN = r'(?:✅|❌)\s*(GREEN|RED)'` (case-insensitive)

     - Extract all other fields (entrada, horario, resultado, placar) — each can be None if not found
     - Derive dia_semana from `datetime.now().weekday()` (0=Mon..6=Sun)
     - Return dict: `{"message_id": message_id, "liga": liga, "entrada": entrada, "horario": horario, "periodo": None, "dia_semana": dia_semana, "resultado": resultado, "placar": placar, "raw_text": text}`
