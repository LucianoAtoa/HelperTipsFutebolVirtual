**Secao 3 — Setup Local:**
Instrucoes passo a passo:
1. Clonar o repositorio
2. Criar e ativar virtualenv: `python3 -m venv venv && source venv/bin/activate`
3. Instalar dependencias: `pip install -e ".[dev]"`
4. Copiar `.env.example` para `.env` e preencher credenciais Telegram (API ID, API Hash, Group ID de https://my.telegram.org) e senha do PostgreSQL
5. Criar banco PostgreSQL:
   ```bash
   createdb helpertips
   createuser helpertips_user
   ```
6. Rodar listener: `python -m helpertips.listener`
7. Rodar dashboard (em outro terminal): `python -m helpertips.dashboard`
8. Acessar dashboard em `http://localhost:8050`
